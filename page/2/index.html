「悠」

发表于 2020-03-16 分类于 诗歌

       此集述青年往事，萦萦欢愁，今始理毕，或以为记。

       「其一」
       年尽雪覆城，客无早入梦。
       卿语未及辨，怅醒不知冷。

       「其二」
       聚时竟匆匆，曼歌初动容。
       不知从何语，念轻月蒙胧。

       「其三」
       归风阻隔断，闲云何日谈。
       一夜自在雪，皑皑在山原。

       「其四」
       雾消雨初停，杳语人在听。
       凉州旧词曲，登高望晚晴。

       「其五」
       丛桃往时栽，逢春枝剪裁。
       会知人相候，花开月徘徊。

       「其六」
       初柳绽滴青，木裙风衿娉。
       有时望月光，银原夜中行。

       「其七」
       悄绿花成行，心野陪春光。
       岑仙风吹去，寻踪天空旷。

       「其八」
       人懒事耽搁，时光不留待。
       末诗还春分，今又谷雨来。
       闻说踏初雨，云薄夜不该。
       漫漫绿草原，遍是格桑开。

       「其九」
       何事暮春月如丝，摇杨风絮似相识。
       四月西乡墙外柳，君不在，约未迟。
       『值末春，行在杨絮雪中，蓦然念故人。人已不知所在，情绪也不知何往。原知佳年在少，惜莫蹉跎。笔名一事，尚无绪，且待些日。』

       「其十」
       岭路墨山横，凉人与相逢。
       漫道思绪事，大风云满城。

       「其十一」
       风卷云向西，春山若在即。
       去影一处望，烟月每初七。

       「其十二」
       烟杨画柳一半月，薄暮浅裳凉云叠。
       便纵仰思念似雪，微情怯，春来歇。

       「其十三」
       残红失落雾沾风，迷处寻来原未曾。
       最难料得中夜雨，无声望，忘半生。

       「其十四」
       拾阶细驻窥月色，满树夏来摇微歌。
       如珍如念如流火，相倚絮忆逆长河。

       「其十五」
       午梦有芦管，轻声云转寒。
       方才悠念起，过风推纱帘。

       「其十六」
       遥雷翻墨岭，骤雨夜半停。
       如何微心已，夏来语冰凌。
       『月余未寄，复习故耳。今诸试皆已，始得说来。念此月间落雨逐离人，微为其伤。不奈时光冷寂，落索丝凉。远行者远，云天无迹，使念者怀往，方寸间笑也哭也。』

       「其十七」
       一簇亮蝉匿深暑，半午晴光倦睡足。
       方才沾水清凉有，依稀片梦寻无处。
       『时光逸弋，半夏已疏。学期终，似有事未能尽。每梦里仿佛心明，梦醒反寻踪无处。喟叹浮情无所依附，或人命微，有时想起，有时忘记。』

「诸」

发表于 2020-03-15 分类于 诗歌

       此集强作词，述学后诸事杂感，寥寥数言，以为自言。

       「清明」
       浑然夜风，藏云推月照离人。明日西山，几家役青坟。
       花渐草染，咸心在暮春。拭前尘，回看烟村，来径已无痕。

       「初夏」
       风浮小栽绿意，兼听夜雨点滴。但梦折梧年，寻常事。
       轻花止絮相似，萱洲杏月在西。今人各几何，多别时。

       「端阳」
       有雨半旬渐蕃秀，初晴惹众游。曲弄喧街，悬艾市酒，饮同大夫愁。
       斯人如兰一朝折，不可留，未肯休。芷岸清波，长云向晚，今昔月影稠。

       「迟语」
       别后期年笺代语，未闻款歌巷愈芜。相去海内知何处。
       好风陌上与歇足 ，丽日花田人恍惚。只今夜听雨嘱咐。

       「长夏莫忆」
       烟山雨渡，画中执伞两立处。若即还离，只字终难溯往昔。
       留觞取水，道是千情饮不醉。草侧花黄，云晚夏长月照江。

       「中秋」
       雨未云已散，迷离秋后原。久在山间心自闲，凭高谁哦水调、月中天。
       盈盈欲相问，脉脉不语言。坐里恍惚觉又年，怅然若将一饮、缈似烟。

       「亭间」
       烟川外，日薄山欲颓。曲廊不知何时作，尽撒离别千斛泪。饮罢谁人回。
       未已矣，含怅云处佪。檐瓦未期骤雨落，银铃叮咚止轻寐。哪堪夜梦归。

       「小雪后」
       小园暮色，霜花落，瑟瑟。扫却奈何情多。年年也，春潮夏泼，往来客。
       留觞饮、陈词付墨。杳湖河。离欢向无良策。若卿语、为云几何。月如昨。

       「行三」
       听闻归溪乡，几番故事梦里长。东邻米酿西家渔，忆相。尚是无忧少年郎。
       父兄扫明堂，斗饮韶华青丝霜。奈何年年要别去，轻怅。夜雨初晴半月上。

       「少年游·谷雨后湟」
       携游丹城半古今，隍楼钟鼓频。笺红柏香，愿祈轻灵，长善皆有信。
       一衫春雨百花临，凭风青丝琴。山稠地远，凉云骤行，无物心自欣。

       「千秋岁引·夏至满来」
       风催浮絮，花再满枝，相与午后约未迟。提将旧事劝新饮，怅把故人怀微思。红衣裳，浅笑靥，依稀识。
       巷子明月少观者，楼台画柳多有诗，盈缺荣枯不曾止。湖鱼两忘始自在，奈何当初甚迷离。且卧听，大乎小，雨点滴。

       「御街行·端阳二事」
       封香裹粽彩线萦。担壶食，陌上行。酒酣偏用巨杯饮，怀醉相话近情。室有所养，闲资可娱，乐得称清平。
       暮落湖光南山屏。三两事，俱云形。一舟苹花懒去扫，沾衣雨细风轻。往夏悠而，翻书正在，佳人如梦令。

「欢」

发表于 2020-03-14 分类于 诗歌

       人生如梦，为欢几何。

       「夏至而莫失」
       雨慢花时令，絮止夏蕃青。
       渐语二三事，言淡怅若影。
       昔年书信遥，相望一盈盈。
       俗生余大梦，轻骑堪寻卿。

       「暑日好梦」
       好梦一场欢，半醒酒尚干。
       拜月长连理，佳期几日还。
       烟愁胡不往，凉云缘何散。
       那时萍水客，如今向青山。

       「不计」
       悠而夏三月，翻书惘在今。
       鹦骨有埋处，猫形无迹寻。
       执意常自扰，情私每被擒。
       未若不相遇，晏游各从心。

       「咸心」
       今朝与明日，再饮风浅时。
       花繁寻常落，人凉归去辞。
       为欢怅几许，造梦念成痴。
       一觞凡尘酒，醉笑她不知。

       「初见」
       盈盈初相见，怔怔少语言。
       无故心慌乱，窃看尔眉间。
       天公且作美，新妆怕雨沾。
       轻声道别去，浮香随车远。

       「今非」
       昨日飞絮梦，今非少年郎。
       孤舟偶有客，云别每无常。
       与我独归酒，深醉复何妨？
       天涯随处是，此心适苍莽。

       「吾道姑友」
       轻衫惹春潮，负剑终难抛。
       新娇有心问，院外谁在瞧。
       故梦三千景，不过偶同道。
       叹息几人知，未凉桂花糕。

       「遥寄老九兄怅然一饮醉又何妨」
       煌城孤泉外，难为君独酌。
       几番对饮愿，千山每阻隔。
       烟愁又怎讲，岁风自寥落。
       花凉谁入梦，云轻似她歌。

       「白露」
       白露青山日渐凉，云迹亦匆忙。
       仓皇雨落，何处能藏，仰面尽杯觞。
       三分醉意七分想，歌哭又一场。
       无言相念长，风白忧伤，月出照卿裳。

「四」

发表于 2020-03-13 分类于 诗歌

       四字不成诗，只片语而已。

       埋云深处，惊一回首

       坛城烟月，落无风催

       云栈希岑，卫事越奔

       疏年可忘，寒阡明晰

       舟而雨下，春也无涯

       絮日繁花，奔塵雨馬

       涼云之隙，浮櫓未遲

       煙山雨渡，草側花黃

       山火佳月，只字情些

       流嵐相與，暮靄雲和

       焉知煙月，在彼初弦

       別來只雪，眉字春些

       行者自在，落雨无它

       夜郎之西，月影轻骑

       曾道而今，怀觞壶饮

       风尘以往，究竟云开

       双城夏晚，宁梦轻安

       尽夜花雨，泠然我衣

       萤流星瀑，云何不孤

       月白雨骤，之子无忧

       烟长月谧，秋裳相宜

       寒谈时光，相思怎讲

       云轻半夏，落梦而秋

       灵台七处，烟波一影

       飞花可怅，夜雨温柔

「旅时」

发表于 2019-11-12 分类于 诗歌

      窗棂外漫白月光，照雪
      寒渐，晴来风息，夜而清绝
      细思碎想，久未成章，道是匆忙
      却浮梦幽幽，忆彼藏行
      红檐白瓦，川流步街
      才急雨斜风，又人面桃花
      松湖如玉，夜月为眸
      惜杳杳闲时，倚肩相看好
      想来某日，你我有桃林一片
      静待春风，黄花梅酒歌翁媪

「秋年」

发表于 2018-10-03 分类于 诗歌

       霜林晓月半城烟，书香酒浅一日闲。
       非卿笔落花笺语，茫寐何期已经年。
       相识无枉春风面，人生大梦且笑颜。
       原知此情多艰险，会以两心拨云间。

「太平年•此秋」

发表于 2018-09-24 分类于 诗歌

       落落秋风飒飒雨，何妨杯觞举。
       人间花木两醉眼，荣枯一浮烟。
       往来世事本俱无，此心是安处。
       有皎皎月明，云上轻行。

「看云去」

发表于 2018-08-22 分类于 散文

       天上云生云没，如人世大境，得见有无聚散，悲喜离合。

       春来，山风轻暖，新绿拾阶而上。山下林场封闭，只有少许几个虔诚香客得以通行。所携黄纸篆香焚作青烟，油粮淡酒供养道人。拜祝始毕，问所求事，不外生老病死，嫁娶婚丧。对以福由心生，神明已知，则其心其情自有所归。或有闲人仅以山色清绝，意在登临而凭高一啸，则亦有所快。唯一老道自谨，和沙拌泥，修葺厢屋，虽汗流气粗，而颜色无改。相问道法，对以自然，若天中云，随风舒卷，无多一心。

       喜见夏花，飘扬清香满是。亦如初初见，眉间笑意浅。引指唤蝴蝶，桃花落处，暮色里轻声别。群山有黄庙，琉璃金顶，映绿柳白杨。幢幢经殿，游人如织，信者其半，燃灯捐钱，乞祈福安。午后疏雨落，殿外经筒未止，殿内柏香沉沉经久，静中卿亦燃灯：想来求此念彼，不若一祝苍生罢。

       沙海浩瀚，渺然一舟。不知山上岩石，经几年霜雪才得风化。而湖水青青，远接天色，似乎住着神灵。不料风急雨骤，小鼠在草丛中的家门口看人车往来匆匆。人们的匆忙总是坏事，让马儿不得纵蹄，让牛儿不得闲刍，甚至草地都不得重新露头。于是，匆匆复匆匆，一生都在匆匆。

       如果可以，当风停雨住，阳光重新照在湖面，我要看着你的眼，在你像湖水的眼里，寻找清澈的时间——那时间里有你我相约看云去。
