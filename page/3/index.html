「雨记」

发表于 2018-04-25 分类于 诗歌

       此晨梦轻，轻叩窗是雨点滴。
       山色始绿，有薄烟青障，白鸟一双入云。
       院外飞花无算，青石路，俱粉红。
       前日卿谓桃花者也。
       风儿且过，落瓣踟蹰，不敌春日急。
       如闻纤步细碎，袅袅向尘去。
       余我一杯淡酒，浮花间掠影，怅天色若明。
       昨夜梦来对饮，昔时种种难尽。
       造化岁月，才见盈盈笑靥，又将新妆泪染。
       人生几回离别，春有繁花冬有雪。
       客尘雨冼，不日来见。

「风马」

发表于 2018-03-30 分类于 散文

       云梦轻扬，鱼儿在天空镜像成双，盈盈相望，你的温柔，只怕在我的远方。

       檐铃随风儿叮咚咚响，正梦醒时，马蹄声在窗外停下。木板门咿呀打开，新柴才添入灶下，火光已然跃动，感染着壶中夹冰的河水，渐渐欢腾。晨茶清香，似有袅袅形质，浮起在轻尘喧嚣的第一缕灿阳。今日，晴朗。

       村中向阳的小树新芽已发，打马而过时，带起的风稍稍卷起细枝，轻轻拉扯行旅人才换的薄裳。村外灿阳初照，山路弯弯曲长，就那么一时走在光明，一时隐入暗晦。对向的车马匆匆，都来不及看清面孔，便又擦肩，错过。而路旁等待的人们，更多是因为生计奔波而表现的匆忙疲倦。又该在哪里寻觅那张笑脸？

       走过很久，路边出现一棵小树，树上开着小花，花瓣只有指甲大小，满枝满桠，像是落满了安静的粉白蝴蝶。花儿昭示春天，可若不是你婉然一笑，我不会知道春天最先藏在你的眼中。长街弄巷，杨柳依依像极了谜语，却没人猜得透这里的百转千回，尔后轻轻守候。

       风马相见，梦昧平生。

「春衫」

发表于 2018-03-15 分类于 散文

       寒轻雨骤，春日来归。

       午后微风轻起，吹皱了云裳，让出一片蓝盈盈天。横斜的电线组成五线谱，谱上有缺月袅袅转过，几只乌鸦飞来，像是无声流淌的音符。铁塔上悄悄多了一个枯枝搭就的巢，穿黑白礼服的喜鹊先生站在一旁，偶然转动着脑袋，仿佛思索着什么。

       人们的房子依然生着炉火，一冬未擦的玻璃尚留着风雪携泥的印记。遇上勤快的主人，隔着明亮的窗，看得见脸蛋热得红彤彤的小孩子，指着路过的正在换毛的邋遢狗嬉笑。狗子倒不介意，含混饱肚子后，大抵会找一隅向阳地，懒洋洋躺下，半梦半醒间待春树发芽，河水欢畅。猫儿亦安逸，一边打着哈欠一边抱着双爪搓着脏兮兮的脸，不紧不慢地跳下热炕头，随意吃些后出门，在矮墙上跟同伴嬉戏。牛羊还没有新鲜的春草可吃，时常边嚼着去年的干草，边憧憬着撒蹄在青山间的时光。有时它们想得出神，眼睛里透出的光，应该是那蓝白相间的花，有淡淡的香。

       忽忆『看美人头上，袅袅春幡』句，虽那时代已相去甚远，却想有某日来约，亦不妨踏春去。坐在山石上，看轻飘的云影幻变，找寻掠过田间林梢，春风的足迹。对饮一坛昨秋你酿的谷酒，醉眼里说这春日繁华。放飞我做的风筝，在它飞很高的时候放手，那儿时失去的，不会再悲伤。

       时光如河，泛舟其上，水中是你春衫上浅色的花。

       只是舟而雨下，春也无涯。

「初雪的语」

发表于 2017-12-20 分类于 散文

       期盼也好，祈愿也罢，这年的初雪终是盈盈落下。白日里，车来人往，雪声寂寂。到那深夜人静，如行街过巷，则能幻听雪语。

       『猫』

       猫离家的去向总是谜题。也只有在这雪夜里，可以看着那一朵朵轻盈的白色“梅花”探寻。足迹不会说话，沉默地昭彰足迹主人如何来去。由于那深浅与步长都没有规律，也就随意想象它站在楼门口先是被风吹个激灵，眯着睡眼，瑟缩地踏入雪中。脚下一阵冰凉，导致它清醒，而来不及想上一场雪是什么时候的事，路过的汽车已将灯光野蛮照来，只得匆匆踏入草地，藏匿起来。一样藏匿的是草下的老鼠。雪封了洞口，风吹不到，虽然暗无天光，挤挤攘攘却既暖和亦无忧。继续走，花坛的矮墙上它停下，是风吹来路人的话，含混不清，无关悲喜，倒真切的没有麻雀的鼾声。想它们白日为果腹奔波，已累得打不出呼噜。出了大院，是大马路。路旁的楼虽年年渐新渐高，它却从未曾留心，关系所在是楼底守门的黑藏狗。它那扭七扭八的脚印说明它是聪明地试探了拴狗链子长度的，也许心怀颤悸，不过走起来必是神情自满。终于追不上“梅花”，给人设置的大门挡不住它的游历，那么，猫你好运。我期冀着看你迎着阳光，眯着眼在早晨踏雪而归，身影疲惫，面色欢欣。

       『河』

       雪到傍晚已停，天上渐渐分辨灰暗的云影，星光只能在缝隙里析出光亮如豆。附近天空的舞台被沿河的彩灯统领。红黄蓝绿，粉紫青白，羞煞月光。待她躲进稀薄的凉云，沉默的雪松们叹息，抖落身上白雪。桥上，枕木间，一节节铁轨披雪而眠，过去隆隆的岁月，全入梦境。小心不打扰它们的回忆，慢慢踩着石板向西。行至中央，捂着冰凉的栏杆站定。往北，齐整的岸堤陪伴灯柱，灯火又点燃街道，虽然看不了很远，总是安静祥和。河道中央，水结冰，浅雪覆盖，映着岸边灯火，几处暗淡，几处如镜。桥的南边冷寂，层冰下的河水流出，汇集在一起。没了长岸的护送，少了水泥平地的帮忙，它们蜿蜒委曲，坚定不回，只流得更远。满川的白石和落雪，何须言语上鼓励它们，多少个千年，彼此都作着陪伴。当最后一滴河水在最远的地方蒸发或者渗入泥土，并非结束，山顶惊艳世人的雪，宛然只是开始。

       『墙』

       小巷不下雪时，只是宁静。一旦下雪，便添了空灵。当手抚冰冷土墙，轻轻拐入小巷，踏上雪覆过的小路，能感到雪被下的尘土稀松。雪墙坚硬，雪尘柔软，回忆扑面而来。抬脚快乐，放足悲伤，走过的地方，原都有迹可循。多么遥远的时光，也没时光本身古老。不在现实里，在被酒浸渍的纸上。主人公可以一去不回，记下故事的人却必须前往凭吊。那事情的意义无需多么冠冕堂皇，能在这时拿来哼哼上一段不全的曲也是好。纵不提相见，一如未相忘。阿嚏。心作小屋，原石作料，院墙高筑。每孤单失落惶惶无助，倚墙角而泣，自品欢伤，垂泪簌簌。一日墙倾，始知外间天地广阔，再无需倚，自行莽莽处，引四方歌喉。阿嚏。记住，或者忘却，都是美好。

       人在雪后，月是初七。

「巷子」

发表于 2017-12-12 分类于 散文

       天无风无月，只星空流转，像是默片播放。

       别已六年，再一次走进巷子。

       巷口的块石蒙尘而卧，应是冬日罕有闲人来晒日头。碎石小路上偶尔可见主妇洗完衣裳泼倒的污水成冰，留着过路人的鞋印。有些土墙将倾，是几根干枯的木头勉强撑着，裂隙间可见人家后窗透出的几缕昏暗灯光。跨河的小桥拆去，连河水也枯涸。几棵傍水而生的榆树，被两岸新装的铁丝围栏困住，尽折了靠河一面的枝桠。我翻过铁栏，想在河底抓一把细沙，却见到一只张着口的大头皮鞋和一册过时的动物画年历。原来上下游不远的地方，有更大的桥，其上灯火流明，车马匆匆。

       站在你曾寄居的院子门前。遮雨的红瓦破掉两块，露出土的地方有枯草一株，不知何年春夏所生。砖墙的缝隙里满是尘土，只砖面上曾刻的简笔娃娃依稀可辨。大门红色的油漆剥落，锈迹围绕着残破的门神画像。再下面，一柄铜锁轻悄悄拦在两扇门间。

       记否。某春日初雨来急，无伞衣湿，本来狼狈，却因见到门前积雨倒映的天空而欣喜忘忧。某夏晚看烟花归来，于门前闻邻一小哥吟“天上星，亮晶晶，永灿烂，长安宁”一句。某秋后夜，满月斜照，半巷月光，半巷墙影，自有“迈兮音尘阙”之意。某冬日午，大雪纷纷而落，将紫色围巾起舞，翩然如梦事。

       巷子无名，不过恰逢你我。

       世上不缺繁华，也没少了冷落。想来哭笑两可，只是岁月仓皇罢。
